# Cloud-Based College Event Registration System

## 📌 Project Overview
This project is a web-based application that allows students to register for college events such as Tech Fest, Cultural Events, and Workshops. The system collects user input through a form and stores the data in a cloud database.

## 🎯 Objective
- To develop a simple event registration system
- To allow students to register for events easily
- To store registration data securely in the cloud

## 🚀 Features
- Student registration form
- Stores data in cloud database
- Displays success message after registration

## 🛠️ Technologies Used
- Node.js
- Express.js
- HTML/CSS
- Google Firestore

## 🏗️ Architecture
User (Browser) → Node.js Application → Firestore Database

## 💻 How to Run Locally
1. Clone the repository  
2. Run: npm install  
3. Run: node server.js  
4. Open: http://localhost:3000  

## ☁️ Cloud Integration
This application uses Google Firestore as a cloud database. The project is designed for future deployment on serverless platforms.
